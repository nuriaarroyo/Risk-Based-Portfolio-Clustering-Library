# Notebook Layout

This folder is organized into three roles:

- `thesis/` contains the notebooks that support the final thesis narrative and paper-facing visualizations.
- `final_experimental_setup/` is the active workbench for the final experiment analysis.
- `archive/` contains demos, validation notebooks, exploration, and loose image exports kept only for reference.

The current cleanup keeps `final_experimental_setup/read_final_results.ipynb` in place because it is still an active working notebook.
